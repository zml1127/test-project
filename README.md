# 物品管理平台

## 项目概述

一个基于浏览器的物品管理平台，用于帮助用户管理柜子和物品。系统支持用户认证，采用响应式设计，适配手机端和PC端。

## 1. 业务边界

### 1.1 包含范围 (In Scope)

#### 柜子管理
- 柜子的创建、编辑、删除
- 柜子列表展示
- 柜子详情查看
- 柜子照片管理

#### 物品管理
- 物品的创建、编辑、删除
- 物品列表展示
- 物品详情查看
- 物品多照片管理
- 物品类别管理（按位置分类：客厅、卧室、厨房、书房、阳台、卫生间、储物间、其他）

#### 物品归类与移动
- 物品与柜子的归属关联创建
- 物品归属变更
- 物品当前位置标记
- 历史轨迹记录（最近添加物品列表）

#### 全局搜索与浏览
- 按物品名称关键字搜索
- 按柜子名称关键字搜索
- 按柜子位置筛选浏览

#### 仪表盘首页
- 统计卡片（总柜子数、总物品数、最近新增）
- 快捷入口（导航栏）
- 全局搜索框

#### 照片上传与查看
- 相册选择图片
- 图片压缩（Base64编码存储）
- 图片存储
- 图片预览

#### 基础用户认证
- 登录/登出
- 单角色（管理员）访问控制

### 1.2 不包含范围 (Out of Scope)

- **权限细分与审核流程**：普通家庭成员角色、物品修改建议提交、审核机制（标记为V2）
- **二维码生成与扫码**：柜子二维码、物品二维码、相机扫码查找（标记为V2可选扩展）
- **微信小程序端**：V1优先实现Web响应式端，小程序端标记为V2
- **财务与资产估值**：物品价格、购买金额、资产统计、保险信息
- **智能提醒**：保修到期提醒、保质期提醒、维护计划提醒
- **条码/ISBN自动识别**：扫描商品条码自动填充信息
- **数据导入导出**：CSV/Excel批量导入导出（标记为V2可选扩展）
- **多家庭/多租户**：系统仅服务于单一家庭，不涉及家庭间数据隔离

## 3. 功能需求

### 3.1 用户认证系统
- **登录功能**
  - 用户名和密码验证
  - 登录状态持久化（Session）
  - 默认管理员账号：admin / admin123

- **登出功能**
  - 清除登录状态
  - 返回登录页面

- **访问控制**
  - 单角色（管理员）访问控制
  - 未登录用户访问管理页面时跳转登录

### 3.2 仪表盘
- 展示柜子总数统计
- 展示物品总数统计
- 各柜子物品数量分布
- 最近添加的物品列表（历史轨迹）

### 3.3 柜子管理
- 创建新柜子（点击按钮展开表单）
- 设置柜子名称
- 设置柜子所处位置（位置选项：客厅、卧室、厨房、书房、阳台、卫生间、储物间、其他）
- 上传柜子照片（支持图片预览）
- 删除已有柜子（级联删除柜子内物品）
- 查看所有柜子列表（默认显示全部）
- 柜子搜索功能（按名称或位置搜索）
- 每个柜子显示其内部的物品数量
- 显示柜子缩略图

### 3.4 物品管理
- 创建新物品（点击按钮展开表单）
- 设置物品名称
- 为物品选择放置的柜子
- 上传物品照片（支持图片预览）
- 删除物品
- 物品搜索功能（按名称搜索）
- 物品显示其所属柜子的位置信息
- 显示物品缩略图

### 3.5 物品归类
- 创建物品时选择放置柜子，建立归属关联
- 物品记录所属柜子名称和位置
- 删除柜子时自动删除该柜子下的物品

### 3.6 Tab切换
- 仪表盘：展示统计数据
- 柜子管理：创建柜子 + 柜子列表
- 物品管理：添加物品 + 物品列表
- Tab切换时自动重置搜索条件并显示全部数据

### 3.7 表单交互
- 创建/添加按钮点击后展开表单
- 再次点击收起表单
- 提交成功后自动收起表单
- 取消按钮可收起表单并重置

## 4. 技术实现

### 4.1 架构
- 纯静态页面（HTML + CSS + JavaScript）
- 单页面应用（SPA）架构
- 使用LocalStorage进行数据持久化
- 文件分离：HTML结构、CSS样式、JS逻辑分离

### 4.2 用户认证
- 前端模拟认证（基于LocalStorage）
- Session管理机制
- 管理员角色标识

### 4.3 响应式设计
- 移动端优先设计
- 媒体查询适配不同屏幕
- 移动端汉堡菜单
- 桌面端顶部导航栏

### 4.4 照片上传与存储
- Base64编码存储图片
- 支持图片预览
- 文件选择触发预览

### 4.5 搜索功能
- 柜子名称模糊搜索
- 柜子位置筛选
- 物品名称模糊搜索
- 实时搜索结果更新
- 切换Tab时自动清空搜索条件

### 4.6 数据存储
- 柜子数据：cabinets数组（含名称、位置、照片、创建时间）
- 物品数据：items数组（含名称、柜子ID、照片、创建时间）
- 用户数据：users数组（含admin用户）
- Session数据：currentUser

## 5. 用户交互流程

### 5.1 登录流程
1. 用户访问系统 → 显示登录页面
2. 输入用户名密码 → 验证身份
3. 验证成功 → 进入主页面（默认显示仪表盘）
4. 验证失败 → 显示错误提示

### 5.2 主页面交互
- 顶部导航栏（仪表盘、柜子管理、物品管理）
- 仪表盘展示统计数据卡片
- Tab切换显示不同内容区域
- 柜子/物品管理页面包含创建按钮和列表

### 5.3 柜子创建流程
1. 点击"创建新柜子"按钮
2. 填写柜子名称
3. 选择柜子位置（类别）
4. 上传柜子照片（可选）
5. 点击"确认创建"
6. 表单收起，列表更新

### 5.4 物品创建流程
1. 点击"添加新物品"按钮
2. 填写物品名称
3. 选择放置柜子（建立归属关联）
4. 上传物品照片（可选）
5. 点击"确认添加"
6. 表单收起，列表更新

### 5.5 搜索浏览流程
1. 在搜索框输入关键词
2. 实时显示搜索结果
3. 支持按位置筛选柜子
4. 切换Tab自动清空搜索条件

### 5.6 登出流程
1. 点击登出按钮
2. 确认登出
3. 清除Session
4. 跳转登录页面
