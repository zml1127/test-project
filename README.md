# 🗄️ 物品管理平台

一个简洁美观的物品管理系统，支持柜子和物品的增删改查、数据导入导出等功能。

## 🚀 功能特性

### 📊 仪表盘
- 数据概览：柜子总数、物品总数、平均每柜物品数
- 柜子物品分布：查看各柜子的物品数量占比
- 最近添加的物品：显示最近添加的5个物品记录
- 全局搜索：快速搜索物品名称或柜子名称
- 数据导入导出：支持数据的备份和恢复

### 📦 柜子管理
- 创建柜子：添加新的柜子，设置名称和位置（类别）
- 编辑柜子：修改柜子的名称和位置
- 删除柜子：删除柜子及其关联的物品
- 搜索筛选：按名称和位置筛选柜子

### 📦 物品管理
- 添加物品：创建物品并关联到指定柜子
- 删除物品：移除物品记录
- 搜索筛选：按名称搜索物品

### 🔐 用户认证
- 登录系统：保护数据安全
- 默认账号：`admin` / `admin123`

### 💾 数据管理
- **复制当前数据**：一键导出所有柜子和物品数据到剪贴板
- **导入数据**：从JSON数据导入，支持追加或覆盖模式
- 本地存储：数据保存在浏览器localStorage中

## 🎨 界面预览

- 红色渐变主题配色
- 响应式设计，支持移动端访问
- 卡片式数据展示
- 表格化列表展示

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计（含响应式布局）
- **JavaScript** - 功能逻辑（原生JS）
- **LocalStorage** - 数据持久化存储

## 📖 使用说明

### 登录
1. 打开页面后使用默认账号登录
2. 账号：`admin`
3. 密码：`admin123`

### 创建柜子
1. 进入"柜子管理"页面
2. 点击"创建新柜子"按钮
3. 填写柜子名称和位置（类别）
4. 点击"确认创建"

### 添加物品
1. 进入"物品管理"页面
2. 点击"添加新物品"按钮
3. 填写物品名称并选择所属柜子
4. 点击"确认添加"

### 数据备份
1. 进入"仪表盘"页面
2. 点击底部"📋 复制当前数据"按钮
3. 数据自动复制到剪贴板
4. 将数据保存到其他位置

### 数据恢复
1. 进入"仪表盘"页面
2. 点击底部"📥 导入数据"按钮
3. 在弹出的文本框中粘贴JSON数据
4. 选择导入模式：
   - 确定：追加到现有数据
   - 取消：覆盖现有数据
5. 点击"确认导入"

## 💡 提示

- 所有数据存储在浏览器本地，换浏览器或清除缓存会导致数据丢失
- 建议定期使用"复制当前数据"功能备份重要数据
- 导入数据时需要使用正确的JSON格式
